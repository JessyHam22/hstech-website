PROYECTO HSTECH - LISTO PARA VERCEL

1) Descomprime este proyecto.
2) Entra a vercel.com y crea una cuenta.
3) Elige Add New > Project > Upload.
4) Sube la carpeta completa hstech-final.
5) Cuando termine, abre Settings > Domains.
6) Agrega hstech.mx y luego www.hstech.mx.
7) Vercel te mostrará los DNS que debes poner en GoDaddy.
8) En GoDaddy entra a tu dominio > DNS y agrega esos registros.
9) Espera la propagación y prueba tu sitio.
